Aplikasi Kasir Penjualan Sepatu

Visual Basic + MySQL

Tentang Proyek

Toko Sepatu adalah aplikasi kasir desktop yang dibangun menggunakan Visual Basic dan terhubung ke database MySQL melalui phpMyAdmin. Aplikasi ini mempermudah proses transaksi dan pencatatan penjualan di toko sepatu secara offline.


Aplikasi ini memiliki fitur lengkap seperti pengelolaan produk, transaksi pembelian, manajemen stok, laporan penjualan, dan login pengguna.
