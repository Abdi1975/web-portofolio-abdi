Robot Delivery Maze

Simulasi Robot Pengantar Barang dalam Labirin

Web Simulation Project

Tentang Proyek

Robot Delivery Maze adalah aplikasi simulasi berbasis web yang dirancang untuk memperagakan proses pengiriman barang oleh robot dalam lingkungan labirin. Pengguna dapat mengatur posisi awal robot, titik tujuan, serta rintangan atau tembok dalam peta grid.


Simulasi ini menggambarkan bagaimana robot bergerak dari satu titik ke titik lain sambil menghindari hambatan, menggunakan pendekatan sederhana berbasis logika atau algoritma jalur.
