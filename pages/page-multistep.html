Web Multi-Step

Aplikasi Prediksi Populasi dengan Langkah Bertahap

Web Application

Tentang Proyek

Web Multi-Step adalah aplikasi berbasis web yang dirancang untuk membantu pengguna memprediksi pertumbuhan populasi menggunakan metode numerik berbasis multistep seperti Adams-Bashforth dan Moulton.


Website ini menggunakan pendekatan bertahap (step-by-step) yang memandu pengguna dari proses input data hingga analisis hasil prediksi dengan tampilan antarmuka yang interaktif dan responsif.

Fitur Utama Aplikasi

📝

Input Data Dinamis

Pengguna dapat memasukkan data populasi dari tahun ke tahun secara fleksibel melalui textarea.

🔢

Parsing & Validasi Data

Data yang dimasukkan akan diproses menjadi array numerik yang siap dihitung secara otomatis.

📈

Prediksi Multi-Step

Menggunakan metode Adams-Bashforth dan Moulton untuk memprediksi populasi beberapa tahun ke depan.

📊

Visualisasi Data

Hasil prediksi ditampilkan dalam bentuk grafik interaktif menggunakan Chart.js.

⚙️

Kontrol Fleksibel

Tombol Parse, Clear, dan Predict memudahkan pengguna dalam mengatur proses secara mandiri.

🔍

Analisis Hasil

Disediakan ringkasan metode, tingkat kepercayaan, dan perbandingan hasil antar metode numerik.
